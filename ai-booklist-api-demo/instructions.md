# Instructions

- Keep the code simple and beginner-friendly
- Use plain TypeScript with `tsc`
- Do not use Vite or any bundler.
- Never use `any` type
- Use strict TypeScript
- Use `ESNext` for `target` and `module`
- Use modern ES modules.
- Use descriptive or meaningfull variable, function and class name
- Create a clean, minimal design
- Use latest css framework Bootstrap 5
